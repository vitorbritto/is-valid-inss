# is Valid INSS Number

Validation for INSS Number input.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
var inssInput = getElementById(#contact-inss);
isValid.inss(inssInput);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
